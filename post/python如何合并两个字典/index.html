Python如何合并两个字典

Thu, Jul 25, 2019

Python 3

x = {&#039;a&#039;: 1, &#039;b&#039;: 2}

y = {&#039;b&#039;: 3, &#039;c&#039;: 4}

z = {**x, **y}

Python 2

z = dict(x, **y)
