Python使用datetime来判断近七天

Thu, Jul 25, 2019

[TOC]

strptime

使用strptime来格式化字符串

datetime.datetime.strptime(&quot;2019-10-02&quot;, &quot;%Y-%m-%d&quot;)

# datetime.datetime(2019, 1, 2, 0, 10)

timedelta

使用timedelta来进行计算天数

a = datetime.datetime.strptime(&quot;2019-10-02&quot;, &quot;%Y-%m-%d&quot;)

b = datetime.datetime.strptime(&quot;2019-10-01&quot;, &quot;%Y-%m-%d&quot;)

a - b

# datetime.timedelta(1)

(a - b).days

# 1
