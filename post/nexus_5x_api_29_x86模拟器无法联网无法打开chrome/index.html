Nexus 5X API 29 x86模拟器无法联网，无法打开chrome

Wed, Feb 5, 2020
  1. 在Android Studio中打开AVD Manager
  2. 点击Create Virtual Device…按钮

无法联网的原因没找到，但是无法打开chrome视乎是应为用了Play Store套件，选择没有Play Store的虚拟机即可，亲测有效
